# Notes App

A simple, web-based note-taking application. It lets users create, save, and view notes directly in the browser without any backend using the browser's local storage.

## Features

- Create new notes via a clean form in the browser.
- Store notes locally in the browser.
- 100% client-side, no server or database required.
- Lightweight and easy to extend.

## Technologies Used

- Vite as the build tool and dev server
- React for building the UI
- JavaScript for application logic
- Local Storage for saving notes

## Usage

1. Clone the repository

```bash
git clone https://github.com/ericstober/notes-app.git
```

2. Navigate to the project directory

```bash
cd notes-app
```

3. Run the development server and open in browser

```bash
npm run dev
```
